Javascript API's

Introduction

A Javascript API which collects and produces a list of Studio Ghibli movies, which includes the title and a short synopsis.

Features

Displays all movies by Studio Ghibli utilising their own API.

Collects data from API, sorts data into movie categories, presents data.

Assigns the Studio Ghibli data movie title and movie synopsis, aswell as assigns a background color to the display panel.

Project : Studio Ghibli API

Created for beginners to explore API's as a free resource for learning Javascript.
